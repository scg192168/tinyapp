## Getting Started

- Install all dependencies (see below) using the `npm install` command.
- Run the development web server using the `npm start` command.

## How to Use

- Run your node server
- Browse to `http://localhost:8080/`
- Enjoy all its features!

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- mocha (for tests)
- chai (for tests)
